Actra

Actra is an agentic governance and runtime policy engine for AI agents, APIs, and workflows.

It allows developers to define and enforce policies that control what actions AI systems can take before execution.

Core capabilities:
- Runtime policy evaluation
- AI agent guardrails
- Tool and API access control
- Deterministic enforcement

Actra can be used:
- As an in-process SDK
- As an external MCP proxy for zero-code enforcement

Key concepts:
- Agentic governance
- Runtime enforcement
- AI safety and control
